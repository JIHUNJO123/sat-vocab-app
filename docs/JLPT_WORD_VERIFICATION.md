# JLPT 단어 검증 가이드

## 현재 데이터의 JLPT 관련성

### AnchorI/jlpt-kanji-dictionary 데이터
- ✅ **JLPT 레벨 명시**: 각 단어에 N5, N4, N3, N2, N1 레벨이 명확히 표시됨
- ✅ **JLPT 관련 단어**: 레벨 정보가 있으므로 JLPT 시험에 관련된 단어로 볼 수 있음
- ✅ **신뢰도**: 높음 (JLPT 레벨 정보 포함)

### 데이터 특성
- **타입**: 한자 중심
- **레벨 분포**:
  - N5: 81개
  - N4: 170개
  - N3: 370개
  - N2: 380개
  - N1: 1,135개

## JMdict의 한계

### JMdict는 일반 일본어 사전
- ❌ **JLPT 레벨 정보 없음**: JMdict 자체에는 JLPT 레벨이 표시되지 않음
- ❌ **필터링 필요**: 약 200,000개 단어 중 JLPT 관련 단어만 선별해야 함
- ⚠️ **정확도 불확실**: 어떤 단어가 JLPT에 나올지 판단하기 어려움

### JMdict 사용 시 필요한 작업
1. JMdict에서 단어 추출
2. 검증된 JLPT 빈출 단어 리스트와 매칭
3. 매칭된 단어만 사용
4. 작업량 많음, 정확도 불확실
