# GitHub Pages 설정 확인 체크리스트

## 파일 업로드 완료 ✅
파일들이 저장소 루트의 `docs` 폴더에 올바르게 업로드되었습니다.

## GitHub Pages 설정 확인

1. **저장소로 이동:** https://github.com/JIHUNJO123/jlpt-step-apps
2. **Settings 클릭**
3. **왼쪽 메뉴에서 "Pages" 클릭**
4. **Source 설정 확인:**
   - Branch: `main` 선택
   - Folder: `/docs` 선택
   - Save 클릭

## 활성화 대기 시간
- GitHub Pages가 활성화되기까지 **1-5분** 정도 걸릴 수 있습니다
- 설정 후 잠시 기다렸다가 URL을 다시 확인하세요

## 테스트 URL
설정이 완료되면 다음 URL들이 작동해야 합니다:

- https://jihunjo123.github.io/jlpt-step-apps/privacy-policy-n5-n3.html
- https://jihunjo123.github.io/jlpt-step-apps/marketing-n5-n3.html
- https://jihunjo123.github.io/jlpt-step-apps/support-n5-n3.html

## 문제 해결
여전히 404가 뜬다면:
1. Settings > Pages에서 Source가 `/docs`로 설정되어 있는지 확인
2. 브랜치가 `main`인지 확인
3. 파일이 `docs` 폴더에 있는지 GitHub 웹에서 확인
4. 몇 분 더 기다린 후 다시 시도
