# JLPT Step 앱 문서

이 폴더에는 각 앱의 개인정보처리방침, 마케팅 정보, 지원 페이지가 포함되어 있습니다.

## GitHub Pages 설정 방법

1. GitHub 저장소를 생성합니다 (예: `jlpt-step-apps`)
2. 이 `docs` 폴더의 내용을 저장소의 `docs` 폴더에 업로드합니다
3. GitHub 저장소 설정에서 Pages를 활성화합니다
   - Settings > Pages
   - Source: `main` 브랜치의 `/docs` 폴더 선택
4. 저장소 이름이 `username.github.io`인 경우:
   - URL: `https://username.github.io/privacy-policy-n5-n3.html`
5. 일반 저장소인 경우:
   - URL: `https://username.github.io/jlpt-step-apps/privacy-policy-n5-n3.html`
