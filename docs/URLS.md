# JLPT Step 앱 URL 목록

GitHub Pages를 설정한 후 아래 URL을 사용하세요.

## GitHub Pages 설정 방법

1. GitHub에 새 저장소 생성 (예: `jlpt-step-apps`)
2. 이 `docs` 폴더의 모든 파일을 저장소에 업로드
3. 저장소 Settings > Pages에서:
   - Source: `main` 브랜치의 `/docs` 폴더 선택
   - Save
