# JLPT 단어 생성 스크립트 사용법

## 현재 상태
- 기본 스크립트가 실행되어 각 레벨별 목표 개수의 단어 파일이 생성되었습니다.
- 하지만 현재는 샘플 단어들을 반복하여 확장하는 방식입니다.

## 더 많은 단어 추가 방법

### 방법 1: 스크립트에 직접 추가
`generate_jlpt_words.py` 파일의 `JLPT_WORDS` 딕셔너리에 더 많은 단어를 추가하세요.

형식:
```python
("단어", "품사", "영어 정의", "일본어 예문", "한국어 정의", "한국어 예문", "중국어 정의", "중국어 예문"),
```

### 방법 2: CSV 파일에서 가져오기
공개된 JLPT 단어 리스트 CSV 파일을 사용할 수 있습니다.

### 방법 3: 공개 데이터 활용
- JLPT 공식 문제집의 단어 리스트
- 신뢰할 수 있는 일본어 학습 사이트의 단어 목록
- 오픈소스 JLPT 단어 데이터베이스

## 번역 품질
- 현재는 내장 번역(임베드 번역)을 사용합니다.
- API 없이 미리 번역된 데이터를 포함합니다.
- 더 정확한 번역이 필요하면 전문 번역 서비스를 활용하세요.

## 실행
```bash
python scripts/generate_jlpt_words.py
```
